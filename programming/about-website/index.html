# 何謂 Hexo

Hexo 是一個基於 Node.js，用於建立靜態網站的開源框架。
它允許使用者使用 Markdown 語法編寫內容，然後將其轉換成靜態網頁。

# Hexo 的特點

  1. Markdown 支持： 使用者可以使用 Markdown 語法編寫內容，使得撰寫文章變得簡單且易於理解。

  2. 快速： Hexo 生成的是靜態網頁，這使得網站載入速度非常快，並且更容易被搜尋引擎索引。

  3. 擴展性： Hexo 提供豐富的插件機制，使得用戶能夠根據自己的需求擴展功能。

  4. 命令行操作： 使用者可以透過命令行輕鬆地創建、部署和管理網站。

  5. 佈景主題： Hexo 支援不同的佈景主題，使用者可以選擇適合自己風格和需求的佈景主題。

# 如何安裝

  1. Hexo 本體的安裝請參考 hexo 官網
  2. Hexo 主題的安裝請參考 hexo shoka
